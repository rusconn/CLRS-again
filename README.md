## コード

書中の擬似コードとは別に、実行可能なコードとして TypeScript を書く。

抽象化やインターフェース整備等のソフトウェア工学的なことには注力しない。

標準 API やサードパーティーの API 等は出来るだけ使わない。

## セットアップ

[Deno](https://deno.com/runtime)を使う。
